## 📌 Об проекте

Симулятор чачки - это **текстовая игра** об придуманном питомце под именем "Чачка" Написанная на Python 3.14

## 📌 текущие результаты

Версия игры: v0.00-3
Полноценно играть: ❌ **невозможно**

## ❌ недостатки
* ❌ нету главного цикла игры
* ❌ нету энергии
* ❌ нету способности перемещения чачки
* ❌ и много остального

## ✅ возможности
* ✅ чачка умеет Викать и Кричать и кушать
* ✅ у чачки есть Хп и еда
* ✅ есть обработка методов при мертвой чачке
* ✅ X, Z и карта
