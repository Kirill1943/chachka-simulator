## 📌 О проекте

Симулятор чачки - это **текстовая игра** об придуманном питомце под именем "Чачка" Написанная на Python 3.14

## 📌 текущие результаты

Версия игры: v0.01
Полноценно играть: ❔**частично**

почему Частично?:
* Реализован только выход
* Так-же реализована отрисовка карты и информация о чачке

## зависимости

* rich - библиотека для цветного текста, прогресс баров и различных вещей, импользуется для цветного вывода

## ❌ недостатки
* нет GUI
* читы неполные

## ✅ возможности
* ✅ чачка умеет Викать и Кричать и кушать и перемещаться
* ✅ у чачки есть Хп и еда
* ✅ есть обработка методов при мертвой чачке
* ✅ X, Z и карта
* ✅ есть стамина у чачки
* ✅ есть эффекты (моментальная регенерация)
* ✅ собственная отрисовка карты по команде
* ✅ базовое внутриигровое время
* ❔ имеется основа для читов

## запуск симулятора

* **Запустить игру:** `python start.py`
* **Запустить игру (читы):** `python start.py --cheats`
* **Установить зависимости:** `python manage.py install`
